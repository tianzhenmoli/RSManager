# RSManager
Resource management by python3 and django

# 资源管理器主要功能介绍
（1）图片美术资源管理查看
（2）多目录及文件打包下载
（3）PSD文件生成PNG图片查看
（4）多层目录结构树形展示（非递归，加速页面打开）
